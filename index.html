ULTIMATE DIMENSIONS CHEAT SHEET

1. The "Exceptions & Traps" (Must Know)

Length: Radius, AU, Parsec, Light Year = $[L]$ BUT Angular Displacement = $[M^0L^0T^0]$
(Dimensionless)
Velocity: Speed, Avg/Terminal Velocity = $[LT^{-1}]$ BUT Angular Velocity = $[T^{-1}]$
Acceleration: Linear, Intensity of gravitational field (g) = $[LT^{-2}]$ BUT Angular Acceleration = $[T^{-2}]$
Force: Thrust, Drag, Normal Rxn, Tension, Weight = $[MLT^{-2}]$ BUT Surface Tension & Spring Constant = $[MT^{-2}]$
(Force/Length)

2. Standard Dimension Groups

  • Work, All Energies, Torque, Couple: $[ML^2T^{-2}]$
  • Pressure, Stress, All Modulus (Young's, Bulk, Shear, Modulus of Rigidity): $[ML^{-1}T^{-2}]$
  • Momentum & Impulse: $[MLT^{-1}]$
  • Power: $[ML^2T^{-3}]$
  • Density: $[ML^{-3}]$

3. Principle of Homogeneity

  • Rule 1: You can only add, subtract, or equate quantities with the SAME dimensions.
    Ex: $L + L = L, \quad L - L = L$.
  • Rule 2: You CAN multiply or divide physical quantities with different dimensions to create new ones (e.g., Velocity = Length/Time).

4. Pro-Tips: The Dimensionless Hacks

The Golden Rule of Functions:
The outputs AND arguments of the following functions are ALWAYS dimensionless $[M^0L^0T^0]$:
  • Trigonometric Ratios: $\sin(\theta)$, $\cos(\theta)$, $\tan(\theta)$
  • Exponential Functions: $e^x$
  • Logarithmic Functions: $\log(x)$, $\ln(x)$
Exam Application: In equation $y = a \sin(kt)$, the argument $(kt)$ must be dimensionless. Therefore, $[k] = 1/[t] = [T^{-1}]$.

5. High-Frequency Exam Constants

Constant Formula Dimension
Gravitational Const (G) $F = G \frac{m_1 m_2}{r^2}$ $[M^{-1}L^3T^{-2}]$
Planck's Const (h) $E = h\nu$ $[ML^2T^{-1}]$
Coeff. of Viscosity ($\eta$) $F = 6\pi\eta r v$ $[ML^{-1}T^{-1}]$
Boltzmann Const (k_B) $E = k_B T$ $[ML^2T^{-2}K^{-1}]$

6. Same Dimension Pairs (Time Savers)

  • $[ML^2T^{-2}]$: Work, Energy, Torque, Heat.
  • $[ML^{-1}T^{-2}]$: Pressure, Stress, Young's Modulus, Energy Density.
  • $[MLT^{-1}]$: Linear Momentum, Impulse.
  • $[ML^2T^{-1}]$: Angular Momentum, Planck's Constant ($h$).
  • $[T^{-1}]$: Frequency, Angular Velocity, Velocity Gradient, Decay Constant.
